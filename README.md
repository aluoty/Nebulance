# Nebulance
A 3D space RPG game in development, using React, Threejs, TS(X)/JS, HTML and CSS.

## Ship Controls
+ W for forward steer
+ S for backward steer
+ A for left steer
+ D for right steer
+ Q for upward steer
+ E for downward steer
+ &#8592; for left/anticlockwise turn
+ &#8594; for right/clockwise turn
