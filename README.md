# Nebulance
A 3D space RPG game in development, using React, Threejs, TS(X)/JS, HTML and CSS.

## Ship Controls
+ W for forward steer
+ S for backward steer
+ A for left steer
+ D for right steer
+ Q for left/counterwise roll
+ E for right/clockwise roll
+ &#8592; for left pivot turn
+ &#8594; for right pivot turn
+ &#8593; for up steer
+ &#8595; for down steer
+ Shift for boost
+ Space for laser shoot
